# HK Flow Tape 中文说明

HK Flow Tape 是一个用于 Codex 的港股资金盘面判读 Skill。它的核心目标不是判断一家公司“好不好”，而是先把港股交易里的不同资金层拆开，回答一个更具体的问题：

> 这只港股现在到底是谁在买、谁在卖，资金是否已经从左侧转向右侧？

它适合和更完整的港股投资框架一起使用，例如 `$yunlei-hk`。`hk-flow-tape` 负责资金归因和盘面结构，投资框架负责资产分类、估值和最终仓位动作。

## 适用场景

你可以在 Codex 中这样调用：

```text
$hk-flow-tape 看一下腾讯、阿里、美团、小米、港交所这几天资金是否右侧
```

也可以和港股投资框架一起调用：

```text
$yunlei-hk $hk-flow-tape 看一下持仓港股是否可以买入/加仓
```

典型问题包括：

- 今天南向资金净买入/净卖出哪些港股？
- 股价大涨但南向净卖出，究竟是谁在买？
- 南向资金是否可以认定为从左侧转为右侧？
- 沽空金额、沽空比率、偏离值分别是什么意思？
- SFC 每周可申报空头仓位是否代表空头压力？
- ADR 折价、港股上涨、南向卖出三者同时出现时该怎么解释？
- ETF 大额交易能不能作为个股右侧信号？

## 核心原则

### 1. 不把净额当参与度

这是本 Skill 最重要的规则。

错误口径：

```text
南向净买入额 / 个股全市场成交额
```

这个比例不能代表南向参与度，因为它把“方向”和“交易活跃度”混在了一起。

正确拆法：

```text
南向参与度 = 南向成交额 / 个股全市场成交额
南向方向强度 = 南向净买入额 / 南向成交额
```

其中：

- 南向成交额 = 港股通买入额 + 港股通卖出额
- 南向净买入额 = 港股通买入额 - 港股通卖出额

### 2. 不把沽空成交当净空头仓位

日度沽空数据反映的是当日被标记为空头卖出的成交金额，它可能来自：

- 新增做空
- 套利或对冲
- 做市交易
- 空头回补前后的交易安排

所以：

- 沽空金额和沽空比率代表当日交易压力；
- SFC 每周可申报空头仓位更接近中期空头拥挤度；
- 两者不能混用。

### 3. 不把缺失数据当作零流入

内置的南向脚本使用东方财富 `RPT_MUTUAL_TOP10DEAL` 数据，覆盖的是每天港股通沪、港股通深各自的十大成交活跃股。

如果一只股票当天不在这个表里，只能说：

```text
该股票未被同日十大成交活跃股数据覆盖。
```

不能说：

```text
南向资金为 0。
```

## 判读框架

分析一只港股时，按下面顺序拆：

1. 股价和全市场成交额：上涨/下跌是否有成交支撑？
2. 南向资金：沪港通、深港通分别买卖多少，合计净额多少？
3. 南向参与度：南向交易在全市场成交中占多大比例？
4. 南向方向强度：南向内部是明显净买、明显净卖，还是高换手但方向弱？
5. ADR/离岸资金：ADR 是否折价或溢价，是否与港股走势背离？
6. 沽空：沽空金额、沽空比率、偏离值是否异常？
7. SFC 净空头：周度可申报空头仓位是增加、减少，还是高位回落？
8. ETF/被动资金：指数 ETF 的交易是否能解释个股表现？

最后输出资金状态：

- 已右侧
- 右侧早期
- 分歧
- 仍在左侧
- 只是反弹，还不是反转

## 常见结论模板

### 股价上涨，南向也净买

如果成交放大、南向参与度高、方向强度为正，通常是较干净的右侧信号。

但是否能加仓，还要看估值、仓位上限、公司基本面和价格位置。

### 股价上涨，但南向净卖

不能说“南向右侧”。

这种情况通常说明买盘来自：

- 离岸/外资资金
- 本地资金
- ETF 或被动资金
- 空头回补

交易上更适合持有、观察或做 T，不适合直接用南向右侧作为加仓理由。

### 南向净买，但股价下跌

这是典型分歧：大陆资金在左侧买，离岸或其他资金在卖。

不能仅凭南向净买就判断右侧，需要等：

- 股价止跌；
- 卖出额收敛；
- 沽空降温；
- ADR 不再继续折价扩大。

### 沽空比率高，但净空头仓位下降

这可能是空头回补和交易对冲并存。

如果股价同步止跌放量，可能形成逼空或修复弹性；如果股价继续破位，则仍然是压力信号。

## 安装方式

克隆仓库后，把整个目录复制到 Codex skills 目录：

```bash
mkdir -p ~/.codex/skills
cp -R hk-flow-tape ~/.codex/skills/hk-flow-tape
```

安装后，在 Codex 中使用：

```text
$hk-flow-tape 看一下阿里这几天的资金情况
```

## 内置脚本

### 1. 南向十大成交活跃股

```bash
python3 scripts/southbound_top10.py \
  --date 2026-06-12 \
  --codes 00700 09988 \
  --turnover 00700=129.74 09988=182.29
```

输出包括：

- 港股通沪买入、卖出、净额、成交额；
- 港股通深买入、卖出、净额、成交额；
- 沪深合计；
- 可选的南向参与度；
- 南向方向强度。

### 2. SFC 可申报空头仓位

```bash
python3 scripts/sfc_short_positions.py \
  --history 8 \
  --codes 00700 09988 \
  --shares 00700=9108033432 09988=19193071958
```

输出包括：

- 报告日期；
- 股票代码；
- 可申报空头股数；
- 可申报空头市值；
- 如果提供总股本，则计算空头股数占比。

## 数据源

主要数据源包括：

- 东方财富数据中心 `RPT_MUTUAL_TOP10DEAL`
- SFC Aggregated Reportable Short Positions
- HKEX 沽空成交数据
- 券商终端或行情源的全市场成交额
- ADR 行情与折溢价数据

## 注意事项

- 本 Skill 是研究和交易辅助工具，不构成投资建议。
- 港股资金数据经常存在口径差异，必须标明日期、来源和覆盖范围。
- 实盘交易前必须重新核验实时行情和数据。
- 资金面只能说明“谁在交易”，不能替代公司基本面和估值判断。
