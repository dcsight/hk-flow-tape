# 港股资金盘面判读框架

## 1. 先分层，不要急着下结论

港股个股的涨跌通常不是单一资金解释，先拆成四层：

1. **南向资金**：大陆投资者经沪港通/深港通买卖，偏好会阶段性很强。
2. **离岸/外资资金**：长线外资、对冲基金、本地资金、ADR影响、全球风险偏好。
3. **被动/ETF资金**：指数ETF、成分股调仓、对冲和仓位管理，不一定代表个股基本面判断。
4. **空头资金**：日内沽空与每周净空头仓位，分别代表交易压力和中期拥挤度。

结论必须回答：上涨或下跌主要由哪一层推动？这些层是共振还是背离？

## 2. 关键口径

### 南向资金

- **买入额**：港股通买入金额。
- **卖出额**：港股通卖出金额。
- **成交额**：买入额 + 卖出额。
- **净买入额**：买入额 - 卖出额。
- **南向参与度**：南向成交额 / 个股全市场成交额。
- **南向方向强度**：南向净买入额 / 南向成交额。

不要用 `南向净买入额 / 个股全市场成交额` 代表参与度。这个比例只能表示全市场成交中有多少被南向净方向解释，不能表示南向交易有多活跃。

### 沽空

- **沽空金额**：当日被标记为空头卖出的成交金额。
- **沽空比率**：沽空金额 / 当日总成交额。
- **偏离值**：当日沽空比率 - 近期平均沽空比率；常见口径是30日均值，但必须以数据源定义为准。

沽空金额/比率不是净空头仓位。高沽空可能是新增做空，也可能是对冲、套利、做市或借券交易。

### SFC可申报空头仓位

SFC每周公布聚合后的可申报空头仓位，通常滞后，适合判断中期空头拥挤度：

- 上升：空头仓位增加，或对冲需求上升。
- 下降：空头回补，或风险敞口收缩。
- 高位下降：可能形成逼空/修复弹性，但需要价格和成交确认。
- 低位上升：短期压力增加，但不一定构成趋势性利空。

SFC数据只包含达到申报门槛的仓位，不等于全部空头，也不披露参与者身份。

## 3. 右侧判定

### 已经右侧

同时满足多数条件：

- 股价放量上涨或止跌突破关键压力。
- 南向连续净买入，且方向强度为正。
- 南向成交额在全市场成交中占比较高，说明不是小额净买。
- 沽空比率没有异常上升，或高位空头开始回补。
- ADR/离岸市场不明显背离。

### 右侧早期

- 股价止跌或出现放量反弹。
- 南向由大幅净卖转为小幅净买，或卖压明显收窄。
- 沽空仍高但偏离值下降。
- ADR不再折价扩大。

这类只能试错，不能重仓确认。

### 分歧

典型场景：

- 股价上涨，但南向净卖。
- 南向买入，但股价继续跌。
- ADR大幅折价，但港股被南向托住。
- 沽空高企，同时资金净买。

分歧不是不能交易，而是只能用T或小仓试错，不要直接定义为右侧。

### 仍在左侧

- 股价继续破位。
- 南向和离岸资金共振卖出。
- 沽空金额与比率同步放大。
- SFC净空头仓位上升且价格没有承接。

这类不要因为“跌了两天该反弹”就加仓；只能等卖压衰减和价格结构修复。

## 4. 常见问题的解释模板

### 股价上涨但南向净卖，是谁在买？

优先检查：

1. 总成交额是否放大。
2. 南向成交额占比是否高。
3. ADR/美股同类是否强。
4. 沽空是否下降，是否存在空头回补。
5. ETF/指数是否被大幅买入。

若南向参与度高且净卖，股价仍涨，通常说明离岸资金、本地资金、被动资金或空头回补在接盘。

### 南向净买但股价下跌，说明什么？

这通常是大陆资金左侧买入，对手盘可能是外资/离岸资金或本地资金卖出。不能仅凭南向净买判断右侧，需要看股价是否止跌、卖出额是否收敛、沽空是否降温。

### ETF频繁大额交易说明什么？

ETF大额成交可能来自：

- 指数贝塔仓位调整。
- 对冲或套利。
- 申赎与做市。
- 风格切换，但不必然落到某只个股。

判断个股右侧时，ETF资金只能作为市场温度，不能替代个股南向、ADR、沽空和价格结构。

## 5. 操作分层

- **确认加仓**：资金共振 + 价格确认 + 仓位未超限。
- **试错买入**：资金卖压收缩 + 价格止跌，但未共振。
- **只做T**：资金背离或消息面不明，但波动足够。
- **等待**：南向/离岸/沽空仍未止血。
- **减仓/避险**：价格破位且资金共振卖出。

输出交易建议时必须给出失效条件。没有失效条件的“可以加仓”不合格。
